# AgentOS - Tony & Laura Harvey Real Estate Platform

Complete real estate CRM platform for Premiere Plus Realty serving Southwest Florida.

## Features

✅ All 6 property categories (Residential/Commercial/Industrial × Buy/Rent)
✅ Advanced property search with filters
✅ Agent dashboard for lead management
✅ Customer portal for saved properties
✅ Contact forms and lead capture
✅ Mobile responsive design
✅ Tony & Laura Harvey profiles

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Supabase

## Service Areas

- Naples, FL
- Fort Myers, FL
- Bonita Springs, FL
- Lehigh Acres, FL
